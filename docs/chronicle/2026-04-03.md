### [2026-04-03 06:04:45] Turn 1 〈opus-0403g〉

- **执行**：WebSearch 16 轮，覆盖 OpenAI/Anthropic/Google/DeepSeek/Together AI/Fireworks/Groq/Mistral/xAI/Ollama/vLLM 的 logprobs 支持情况、定价、云部署方案。产出写入 docs/research/2026-04-03-ai-api-cost-comparison.md
- **结果**：完成结构化对比表。核心发现：(1) 计算 perplexity 需要 input/prompt token logprobs，大多数 API 只给 output logprobs；(2) Together AI（echo=True）是最佳托管方案，$1.50/月；(3) Anthropic/Groq 完全不支持 logprobs；(4) 自部署 RunPod+Ollama 约 $15-25/月但最灵活
- **决策**：推荐 Together AI Completions API + Llama 3.1 8B 作为首选方案（$0.10/1M tokens，原生支持 echo=True prompt logprobs）
